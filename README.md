# Remix + Cloudflare Pages

`npm create cloudflare@latest`

- Remix
- Cloudflare Pages
- KV
- D1

## Development

### Migration

```sh
npm run migrations:gen
```

### Start server

```sh
npm run dev
```

Open up [http://127.0.0.1:8788](http://127.0.0.1:8788) and you should be ready to go!

## Deployment

### Deploy

```sh
pages:deploy
```
